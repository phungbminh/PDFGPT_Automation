### Setup
### Installation

1. Navigate to the project directory:

   ```bash
   cd PDFGPTAutomation
   ```
 
 2. Install required packages
    ```bash
    pip install -r requirements.txt
    ```
 3. Run main
    ```bash
    python main.py --folder-input=path --folder-output=path --chrome-path=path --chrome-driver=path
    ```
